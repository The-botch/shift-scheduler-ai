# AIシフトスケジューラー

AIによる自動シフト生成システムのMVPデモアプリケーションです。

## デモ起動方法

```bash
# リポジトリをクローン
git clone https://github.com/info-mnml/shift-scheduler-ai.git
cd shift-scheduler-ai/demo/mvp-demo

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:5173 にアクセスしてデモを開始してください。
