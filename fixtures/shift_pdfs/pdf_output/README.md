## ファイル名の命名規則
- 基本形式: `[改/改2/改3]シフト表_YYYYMMDD-YYYYMMDD_店舗名.pdf`
- 「改」「改2」「改3」は訂正バージョンを示す
- 日付範囲は月初から月末まで
- 店舗名は英語表記
