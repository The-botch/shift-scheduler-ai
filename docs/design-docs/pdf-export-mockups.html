Shift Scheduler AI このシフト表は確定版です

現在の画面表示との対応:

  • • 縦軸: 日付（1日〜31日）+ 曜日
  • • 横軸: 全体サマリー → 店舗サマリー → スタッフ（名前・役職）
  • • セル: 開始時間・終了時間を縦に表示
  • • 色分け: 希望日=緑、NG日=グレー、日曜=赤文字、土曜=青文字

PDF出力内容:

  • • ヘッダー: 店舗名、年月、プラン種別、出力日
  • • テーブル: 現在画面に表示されているMultiStoreShiftTableをそのまま縮小して1ページに収める
  • • フッター: システム名、注意書き
  • • 非表示: 操作ボタン、編集UI、ナビゲーション

1ページに収める方法:

  • • CSS transform: scale() で自動縮小
  • • テーブル幅・高さを計算してA4サイズにフィット
  • • フォントサイズは8-9pxまで縮小可能
  • • スタッフ数が多い場合でもスクロールなしで表示
実装方式の比較
react-to-print 採用
  • + 既存テーブルをそのまま利用
  • + CSSで細かく制御可能
  • + 日本語の問題なし
  • + 軽量（追加パッケージ小）
  • - 印刷ダイアログ経由
2
window.print()
  • + 最もシンプル
  • + 追加パッケージ不要
  • - 「PDFで保存」を選ぶ必要
  • - 制御が難しい
3
jsPDF + html2canvas
  • + 直接PDFダウンロード
  • + 細かいカスタマイズ
  • - 日本語フォント問題
  • - 画像変換で重い
  • - 設定が複雑
実装仕様

パッケージ追加

npm install react-to-print

PDF出力ハンドラー（店舗ごとにループ）

import { useRef, useState } from 'react'
import { useReactToPrint } from 'react-to-print'

const SecondPlanEditor = () => {
  const printRef = useRef(null)
  const [printingStore, setPrintingStore] = useState(null)

  // 店舗ごとにPDF出力
  const handleExportPDF = async () => {
    for (const store of selectedStores) {
      setPrintingStore(store) // 印刷対象店舗をセット
      await new Promise(r => setTimeout(r, 100)) // レンダリング待ち

      const storeName = storesMap.get(store.id)?.name
      await reactToPrintFn({
        contentRef: printRef,
        documentTitle: `shift_${storeName}_${year}_${month}`,
        pageStyle: `
          @page { size: A4 landscape; margin: 10mm; }
          @media print {
            .print-area { transform: scale(0.85); }
          }
        `,
      })
    }
    setPrintingStore(null)
  }

  return (
    <div ref={printRef}>
      {/* printingStoreでフィルタされたテーブル */}
    </div>
  )
}

印刷用CSS

@media print {
  /* 非表示要素 */
  .no-print,
  header,
  nav {
    display: none !important;
  }

  /* 印刷時のみ表示 */
  .print-only {
    display: block !important;
  }

  /* 色を保持 */
  * {
    -webkit-print-color-adjust: exact !important;
    print-color-adjust: exact !important;
  }

  /* テーブル最適化 */
  table {
    font-size: 9px;
    border-collapse: collapse;
  }
}
